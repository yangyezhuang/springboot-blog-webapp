### 简介

> 牛油果博客 是一个基于vue2.x + SpringBoot的博客系统，整体风格为简约型，支持==代码高亮==。项目持续更新，基本功能已实现，有时间会进行页面完善与优化。

<br>

### 技术栈
**前端：**
vue + vue-router + axios + element-ui


**后端：**
springboot + mybatis +shiro

**数据库：**
MySQL
